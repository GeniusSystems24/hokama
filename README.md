# حكماء المستقبل

### **جدول المصادقة**
**الوصف**: يخزن معلومات المصادقة للمستخدمين، بما في ذلك معرف المستخدم وكلمة المرور المشفرة.

- **الأعمدة**:
    - `User_ID`: معرف المستخدم الفريد (INT) `المفتاح الأساسي`
    - `Password_Hash`: تجزئة كلمة المرور المشفرة (VARCHAR)
    - `Password_Salt`: ملح كلمة المرور المستخدم في التجزئة (VARCHAR)
    - `Status`: حالة المستخدم (INT)، مثل 1 لـ "نشط"، 0 لـ "غير نشط"

- **العمليات**:
    - **إضافة** بيانات مصادقة جديدة
    - **تعديل** كلمة المرور المشفرة
    - **حذف** بيانات المصادقة
    - **عرض** بيانات المصادقة للمستخدمين

### **جدول المستخدمين**
**الوصف**: يحتوي على بيانات المستخدمين المسجلين في النظام.

- **الأعمدة**:
    - `User_ID`: معرف المستخدم الفريد (INT) `المفتاح الأساسي والمفتاح الخارجي`
    - `User_Name`: الاسم الكامل للمستخدم (VARCHAR)
    - `User_Image`: رابط لصورة المستخدم (VARCHAR)
    - `User_Phone`: رقم الهاتف المحمول (VARCHAR)
    - `User_Email`: عنوان البريد الإلكتروني (VARCHAR)
    - `User_Gender`: الجنس (VARCHAR)
    - `User_Birthdate`: تاريخ الميلاد (DATE)
    - `User_Rank`: رتبة المستخدم أو نوعه (VARCHAR)، مثل "مدير"، "طالب"، "مشرف"
    - `Status`: حالة المستخدم (INT)، مثل 1 لـ "نشط"، 0 لـ "غير نشط"
    - `Created_Date`: تاريخ إنشاء السجل (DATE)
    - `Updated_Date`: تاريخ آخر تحديث للسجل (DATE)

- **العمليات**:
    - **إضافة** مستخدم جديد
    - **تعديل** بيانات المستخدم
    - **حذف** مستخدم
    - **عرض** قائمة المستخدمين

### **جدول الكيانات**
**الوصف**: يخزن معلومات حول الكيانات المختلفة، مثل المدارس أو الأقسام، بما في ذلك المديرين ومعلومات الاتصال.

- **الأعمدة**:
    - `Entity_ID`: معرف الكيان الفريد (INT) `المفتاح الأساسي`
    - `Entity_Name`: اسم الكيان (VARCHAR)
    - `Manager_ID`: معرف المدير (VARCHAR) `المفتاح الخارجي`
    - `Entity_Phone`: رقم الهاتف المحمول للكيان (VARCHAR)
    - `Entity_Email`: عنوان البريد الإلكتروني للكيان (VARCHAR)
    - `Entity_Image`: رابط لصورة الكيان (VARCHAR)
    - `Status`: حالة الكيان (INT)، مثل 1 لـ "نشط"، 0 لـ "غير نشط"
    - `Created_Date`: تاريخ إنشاء السجل (DATE)
    - `Updated_Date`: تاريخ آخر تحديث للسجل (DATE)
    
- **العمليات**:
    - **إضافة** كيان جديد
    - **تعديل** بيانات الكيان
    - **حذف** كيان
    - **عرض** قائمة الكيانات

### **جدول الطلاب**
**الوصف**: يحتوي على بيانات الطلاب المسجلين، بما في ذلك معلومات الاتصال الأساسية.

- **الأعمدة**:
    - `Student_ID`: معرف الطالب الفريد (INT) `المفتاح الأساسي والمفتاح الخارجي`
    - `Student_Name`: الاسم الكامل للطالب (VARCHAR)
    - `Student_Phone`: رقم الهاتف المحمول (VARCHAR)
    - `Student_Email`: عنوان البريد الإلكتروني (VARCHAR)
    - `Status`: حالة الطالب (INT)، مثل 1 لـ "نشط"، 0 لـ "غير نشط"
    - `Created_Date`: تاريخ إنشاء السجل (DATE)
    - `Updated_Date`: تاريخ آخر تحديث للسجل (DATE)
    
- **العمليات**:
    - **إضافة** طالب جديد
    - **تعديل** بيانات الطالب
    - **حذف** طالب
    - **عرض** قائمة الطلاب بناءً على المجموعة

### **جدول كيانات الطلاب**
**الوصف**: يخزن معلومات حول ارتباط الطلاب بالكيانات، مما يسمح بتعيين الطلاب لكيانات محددة.

- **الأعمدة**:
    - `Entity_ID`: معرف الكيان (INT) `المفتاح الأساسي`
    - `Student_ID`: معرف الطالب (INT)
    - `Status`: حالة الارتباط (INT)، 1 لـ "نشط"، 0 لـ "غير نشط"
    - `Created_Date`: تاريخ إنشاء السجل (DATE)
    - `Updated_Date`: تاريخ آخر تحديث للسجل (DATE)

- **العمليات**:
    - **إضافة** ارتباط جديد بين طالب وكيان
    - **تعديل** حالة الارتباط
    - **حذف** ارتباط موجود
    - **عرض** قائمة الطلاب المرتبطين بكيان معين
    - **عرض** الكيانات المرتبطة بطالب معين

### **جدول المجموعات**
**الوصف**: يخزن معلومات حول المجموعات داخل الكيانات ويسمح بتعيين المديرين للمجموعة.

- **الأعمدة**:
    - `Entity_ID`: معرف الكيان الذي تنتمي إليه المجموعة (INT) `مفتاح أساسي مركب ومفتاح خارجي`
    - `Group_ID`: معرف المجموعة الفريد (INT) `مفتاح أساسي مركب`
    - `Group_Name`: اسم المجموعة (VARCHAR)
    - `Group_Image`: رابط لصورة المجموعة (VARCHAR)
    - `Status`: حالة المجموعة (INT)، مثل 1 لـ "نشط"، 0 لـ "غير نشط"
    - `Created_Date`: تاريخ إنشاء السجل (DATE)
    - `Updated_Date`: تاريخ آخر تحديث للسجل (DATE)
    
- **العمليات**:
    - **إضافة** مجموعة جديدة
    - **تعديل** بيانات المجموعة
    - **حذف** مجموعة
    - **عرض** قائمة المجموعات بناءً على الكيان

### **جدول مجموعات الطلاب**
**الوصف**: يخزن العلاقات بين الطلاب والمجموعات التي ينتمون إليها، مما يسمح للطالب بالانتماء إلى مجموعات متعددة.

- **الأعمدة**:
    - `Entity_ID`: معرف الكيان الذي تنتمي إليه المجموعة (INT) `مفتاح خارجي مركب`
    - `Group_ID`: معرف المجموعة (INT) `مفتاح خارجي مركب`
    - `Student_ID`: معرف الطالب (INT) `مفتاح خارجي`
    - `Link_ID`: معرف الرابط الفريد (INT) `المفتاح الأساسي`
    - `Rank`: رتبة المستخدم أو نوعه (VARCHAR)، مثل "مدير"، "طالب"، "مشرف"
    - `Status`: حالة الارتباط (INT)، مثل 1 لـ "نشط"، 0 لـ "غير نشط"
    - `Created_Date`: تاريخ إنشاء السجل (DATE)
    - `Updated_Date`: تاريخ آخر تحديث للسجل (DATE)
    
- **العمليات**:
    - **إضافة** ارتباط جديد بين طالب ومجموعة
    - **تعديل** الارتباط
    - **حذف** ارتباط
    - **عرض** قائمة المجموعات المرتبطة بكل طالب
    - **عرض** قائمة الطلاب المرتبطين بكل مجموعة
        
### **جدول المجالات**
**الوصف**: يشمل المجالات أو الفئات الرئيسية (مثل الأسرة، التعليم).

- **الأعمدة**:
    - `Entity_ID`: معرف الكيان الذي تنتمي إليه المجموعة (INT) `مفتاح أساسي مركب ومفتاح خارجي`
    - `Domain_ID`: معرف المجال الفريد (INT) `مفتاح أساسي مركب`
    - `Domain_Name`: اسم المجال (VARCHAR)
    - `Domain_Image`: رابط لصورة المجال (VARCHAR)
    - `Domain_Order`: ترتيب المجال (INT)
    - `Status`: حالة المجال (INT)، مثل 1 لـ "نشط"، 0 لـ "غير نشط"
    - `Created_Date`: تاريخ إنشاء السجل (DATE)
    - `Updated_Date`: تاريخ آخر تحديث للسجل (DATE)
    
- **العمليات**:
    - **إضافة** مجال جديد
    - **تعديل** بيانات المجال
    - **حذف** مجال
    - **عرض** قائمة المجالات بترتيب معين
        
### **جدول الأقسام**
**الوصف**: يحتوي على الأقسام الفرعية التي تنتمي إلى المجالات الرئيسية.

- **الأعمدة**:
    - `Entity_ID`: معرف الكيان الذي تنتمي إليه المجموعة (INT) `مفتاح أساسي مركب ومفتاح خارجي مركب`
    - `Domain_ID`: معرف المجال الفريد (INT) `مفتاح أساسي مركب ومفتاح خارجي مركب`
    - `Section_ID`: معرف القسم الفريد (INT) `مفتاح أساسي مركب`
    - `Section_Serial_Number`: رقم تسلسلي فريد (UniqueIdentifier) `مفتاح أساسي ثانوي`
    - `Section_Name`: اسم القسم (VARCHAR)
    - `Section_Image`: رابط لصورة القسم (VARCHAR)
    - `Section_Order`: ترتيب القسم داخل المجال (INT)
    - `Status`: حالة القسم (INT)، مثل 1 لـ "نشط"، 0 لـ "غير نشط"
    - `Created_Date`: تاريخ إنشاء السجل (DATE)
    - `Updated_Date`: تاريخ آخر تحديث للسجل (DATE)
    
- **العمليات**:
    - **إضافة** قسم جديد
    - **تعديل** بيانات القسم
    - **حذف** قسم
    - **عرض** قائمة الأقسام بناءً على المجال

- **الأعمدة**:

    - `Topic_Serial_Number`: رقم تسلسلي فريد (UniqueIdentifier) `مفتاح أساسي ثانوي`
    - `Topic_Name`: اسم الموضوع (VARCHAR)

    - `Topic_Order`: ترتيب الموضوع داخل القسم من المجال (INT)

- **العمليات**:

    - **عرض** قائمة المواضيع بناءً على القسم
        
### **جدول المسابقات**
**الوصف**: يحتوي على بيانات المسابقات، بما في ذلك الأسئلة، الوقت، والمراحل.

- **الأعمدة**:
    - `Entity_ID`: معرف الكيان الذي تنتمي إليه المجموعة (INT) `مفتاح أساسي مركب ومفتاح خارجي مركب`
    - `Domain_ID`: معرف المجال الفريد (INT) `مفتاح أساسي مركب ومفتاح خارجي مركب`
    - `Section_ID`: معرف القسم الفريد (INT) `مفتاح أساسي مركب ومفتاح خارجي مركب`
    - `Topic_ID`: معرف الموضوع الفريد (INT) `مفتاح أساسي مركب ومفتاح خارجي مركب`
    - `Competition_ID`: معرف المسابقة الفريد (INT) `مفتاح أساسي`
    - `Competition_Serial_Number`: رقم تسلسلي فريد (UniqueIdentifier) `مفتاح أساسي ثانوي`
    - `Competition_Name`: اسم المسابقة (VARCHAR)
    - `Competition_Time_Limit`: الحد الزمني (INT)
    - `Competition_Questions_Count`: عدد الأسئلة (INT)
    - `Competition_Stages_Count`: عدد المراحل (INT)
    - `Competition_Stages`: المراحل مع معلوماتها والأسئلة (Json)
    - `Competition_Image`: رابط لصورة المسابقة (VARCHAR)
    - `Competition_Enrichment_Text`: رابط لنص الإثراء (VARCHAR)
    - `Competition_Is_With_Sound`: هل المسابقة مع صوت أم لا (Bit)
    - `Status`: حالة المسابقة (INT)، مثل 1 لـ "نشط"، 0 لـ "غير نشط"
    - `Created_Date`: تاريخ إنشاء السجل (DATE)
    - `Updated_Date`: تاريخ آخر تحديث للسجل (DATE)
    
- **العمليات**:

    - **تعديل** بيانات المسابقة
    - **حذف** مسابقة
    - **عرض** قائمة المسابقات

### **جدول الخطط**
**الوصف**: يخزن معلومات حول خطط الاشتراك المتاحة، بما في ذلك عدد المجموعات والطلاب المسموح بها، ومدة الصلاحية.

- **الأعمدة**:
    - `Plan_ID`: معرف الخطة الفريد (INT)
    - `Plan_Name`: اسم الخطة (VARCHAR)، مثل "الخطة الأساسية"، "الخطة المميزة"
    - `Allowed_Groups`: الحد الأقصى لعدد المجموعات المسموح بها (INT)
    - `Allowed_Students`: الحد الأقصى لعدد الطلاب المسموح بهم (INT)
    - `Duration`: مدة الصلاحية (VARCHAR)، مثل "شهر"، "سنة"
    - `Price`: سعر الاشتراك (DECIMAL)
    - `Description`: وصف مختصر عن الخطة (TEXT)
    - `Status`: حالة الخطة (INT)، مثل 1 لـ "متاحة"، 0 لـ "غير متاحة"
    - `Created_Date`: تاريخ إنشاء السجل (DATE)
    - `Updated_Date`: تاريخ آخر تحديث للسجل (DATE)
    
- **العمليات**:
    - **إضافة** خطة جديدة
    - **تعديل** بيانات الخطة
    - **حذف** خطة
    - **عرض** قائمة الخطط المتاحة
        
### **جدول الاشتراكات**
**الوصف**: يخزن معلومات الاشتراك للكيانات، بما في ذلك حزمة الميزات المتاحة مثل عدد المجموعات والطلاب.

- **الأعمدة**:
    - `Subscription_ID`: معرف الاشتراك الفريد (INT)
    - `Plan_ID`: معرف الخطة (INT)
    - `Entity_ID`: معرف الكيان (INT)
    - `Start_Date`: تاريخ بدء الاشتراك (DATE)
    - `End_Date`: تاريخ انتهاء الاشتراك (DATE)
    - `Subscription_Status`: حالة الاشتراك (VARCHAR)، مثل "نشط"، "منتهي"
    - `Status`: حالة السجل (INT)، مثل 1 لـ "نشط"، 0 لـ "غير نشط"
    - `Created_Date`: تاريخ إنشاء السجل (DATE)
    - `Updated_Date`: تاريخ آخر تحديث للسجل (DATE)
    
- **العمليات**:
    - **إضافة** اشتراك جديد
    - **تعديل** بيانات الاشتراك
    - **إلغاء** اشتراك
    - **عرض** قائمة الاشتراكات للكيانات
        
- **حالات الاستخدام**:
    - عند **إضافة كيان جديد**، يجب إنشاء اشتراك افتراضي له.
    - عند **تعديل كيان**، يمكن تعديل معلومات اشتراكه.
    - عند **حذف كيان**، يجب حذف الاشتراك المرتبط به.
    - التأكد من أن عدد المجموعات والطلاب لا يتجاوز الحدود المسموح بها في الاشتراك.
        
### **جدول مدفوعات الاشتراك**
**الوصف**: يخزن معلومات حول المدفوعات المتعلقة بالاشتراكات، بما في ذلك تاريخ الدفع، المبلغ، وحالة الدفع.

- **الأعمدة**:
    - `Payment_ID`: معرف الدفع الفريد (INT)
    - `Subscription_ID`: معرف الاشتراك المرتبط (INT)
    - `Payment_Date`: تاريخ الدفع (DATE)
    - `Amount`: المبلغ المدفوع (DECIMAL)
    - `Payment_Method`: طريقة الدفع (VARCHAR)، مثل "بطاقة ائتمان"، "تحويل بنكي"
    - `Payment_Status`: حالة الدفع (VARCHAR)، مثل "مدفوع"، "معلق"، "ملغى"
    - `Transaction_ID`: رقم المعاملة من نظام الدفع (VARCHAR)
    - `Status`: حالة الدفع (INT)، مثل 1 لـ "نشط"، 0 لـ "غير نشط"
    - `Created_Date`: تاريخ إنشاء السجل (DATE)
    - `Updated_Date`: تاريخ آخر تحديث للسجل (DATE)
    
- **العمليات**:
    - **إضافة** دفع جديد
    - **تعديل** بيانات الدفع
    - **حذف** دفع
    - **عرض** قائمة المدفوعات المتعلقة بالاشتراكات
